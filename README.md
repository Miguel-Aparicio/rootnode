# rootnode

Proyecto personal de desarrollo de aplicaciones de distinta índole y tecnología.

## Descripción

Este repositorio actúa como nodo raíz de proyectos personales de desarrollo de software, abarcando aplicaciones web, scripts de utilidad, APIs, herramientas CLI y cualquier otro tipo de aplicación en distintas tecnologías.

## Estructura

```
rootnode/
├── apps/          # Aplicaciones completas (web, móvil, desktop, etc.)
├── scripts/       # Scripts de utilidad y automatización
├── packages/      # Paquetes o módulos reutilizables
└── docs/          # Documentación general del proyecto
```

## Tecnologías

El proyecto es agnóstico en cuanto a tecnología. Cada subdirectorio puede emplear el stack más adecuado para su propósito (Node.js, Python, Go, etc.).

## Primeros pasos

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Miguel-Aparicio/rootnode.git
   cd rootnode
   ```
2. Navega al proyecto que te interese dentro de `apps/`, `scripts/` o `packages/`.
3. Sigue las instrucciones de cada subdirectorio para instalar dependencias y ejecutar el proyecto.

## Contribución

Este es un proyecto personal. Si encuentras algo útil, siéntete libre de hacer un fork o abrir un issue.
